# Coding Standards for Hidayah AI
- Use Streamlit session state to manage audio progress and chat history.
- Implement a 'Decision-Based Agent' that routes queries to: 
  1. AlQuran API (Direct verses)
  2. Tavily API (Web research)
  3. FAISS (Local PDF RAG).
- For Urdu support, use RTL (Right-to-Left) alignment in CSS.
